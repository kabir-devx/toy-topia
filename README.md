# 🧸 ToyTopia

**ToyTopia** is a modern, interactive toy browsing and discovery platform built with React and Firebase. Users can explore various toy categories, view detailed toy information, and manage their profiles with secure authentication.

## 🌟 Live Demo

[Visit ToyTopia Live](toy-topia-by-kabir.netlify.app)

## ✨ Features

### Core Features

- 🏠 **Interactive Home Page** - Dynamic toy slider and featured categories
- 🎯 **Category Browsing** - Browse toys by different categories
- 🔍 **Toy Details** - View comprehensive information about each toy
- 🔐 **User Authentication** - Secure login/register with Firebase Auth
- 👤 **User Profile** - Manage user profile and preferences
- 🔒 **Private Routes** - Protected routes for authenticated users
- 🔑 **Password Recovery** - Forgot password functionality
- 💬 **Feedback System** - Users can provide feedback
- 📱 **Responsive Design** - Fully responsive across all devices

### Technical Highlights

- ⚡ Built with React 19 and Vite for blazing fast performance
- 🎨 Styled with Tailwind CSS 4 and DaisyUI
- 🔥 Firebase integration for authentication
- 🎭 Smooth animations with Lottie React
- 🎠 Interactive sliders using Keen Slider and Swiper
- 🌐 Client-side routing with React Router 7
- 🎯 Dynamic page titles
- 📢 Toast notifications for user feedback


### Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (v18 or higher)
- **npm** or **yarn** package manager
- **Firebase account** (for authentication)

## 🛠️ Tech Stack

### Frontend

- **React 19** - UI library
- **Vite 7** - Build tool and dev server
- **React Router 7** - Client-side routing
- **Tailwind CSS 4** - Utility-first CSS framework
- **DaisyUI** - Tailwind CSS component library

### Backend & Services

- **Firebase Authentication** - User authentication and management

### UI & Animation

- **React Icons** - Icon library
- **Lottie React** - Animation library
- **Keen Slider** - Touch slider library
- **Swiper** - Modern mobile slider

### Development Tools

- **ESLint** - Code linting
- **React Toastify** - Toast notifications
